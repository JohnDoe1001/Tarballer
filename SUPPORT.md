# Update Policy

## Supported Versions

Only the latest version and it's derivatives (e.g v1.x) will be supported.

Version name scheme: Version.Feature.Fixes

If a number was omitted, then it means 0.
(e.g 1.2 = 1.2.0; 2 = 2.0.0)

| Version | Supported          |
| ------- | ------------------ |
| 1.1     | Yes                |
| 1.0     | Yes                |
| 1.2     | Yes                |
